Introduction
===========

The Exam Planning System is a web application designed to help universities manage and schedule exams. It provides a comprehensive solution for professors, students, and administrative staff to coordinate exam scheduling.

Features
--------

* User authentication with JWT tokens and role-based access control
* Exam scheduling and management
* Group leader functionality for student representatives
* Course and professor management
* Room allocation and scheduling
* Calendar view for visualizing exam schedules
